## Especificaciones Técnicas

- **Formato**: MP3 (MPEG Audio Layer III)
- **Tasa de bits**: Variable (optimizada por archivo)
- **Frecuencia de muestreo**: 44.1 kHz estándar
- **Canales**: Estéreo para ambiente, mono/estéreo para campanas
- **Duración típica**:
  - Campanas: 3-15 segundos
  - Efectos: 1-10 segundos  
  - Ambiente: 5-15 minutos (optimizado para loop)

## Uso en la Aplicación

Todos estos archivos están integrados en el **Sistema de Cuelist Avanzada** de Sadhana Web v2.0.0:

1. **Acceso directo** desde el editor modal de cues
2. **Categorización visual** por tipo de sonido
3. **Vista previa** antes de añadir a la sesión
4. **Control granular** de timing, volumen y repetición
5. **Combinaciones ilimitadas** para crear secuencias personalizadas

## Organización y Mantenimiento

La biblioteca se mantiene organizada automáticamente:

- **Nombres descriptivos** para fácil identificación
- **Categorización consistente** por tipo de sonido
- **Optimización web** para carga rápida
- **Documentación actualizada** con cada adición
